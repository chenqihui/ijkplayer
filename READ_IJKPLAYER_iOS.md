## IJKMediaPlayer
### IJK的framework库工程  
#### 包含如下目录
    
| 文件夹 | 说明 |
| -------- | -------- |
| IJKMediaPlayer   | 工具、协议及状态等   |
| IJKAVMoviePlayerController   | AVPlayer播放器，参考AVPlayerDemo   |
| IJKMPMoviePlayerController   | MPMoviePlayer播放器   |
| IJKFFMoviePlayerController   | FFmpeg播放器   |
